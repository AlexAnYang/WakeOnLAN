WakeOnLAN - Help

Welcome

Wake On LAN, shutdown software for Windows. A powerful WOL, ping, shutdown, GUI application.
WakeOnLan performs the following tasks

  1. WakeUp a remote computer that is powered off
  2. Shutdown a remote computer
  3. Ping the selected remote computer, to display it's status
  4. Perform an emergency shutdown of ALL defined computers at once
  5. Connect to the remote server via Remote Desktop
  6. Listen for WOL packets (to aid in troubleshooting)
