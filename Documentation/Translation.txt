On this spreadsheet, each page in WOL is on a different tab.
Each string in the program is in the spreadsheet.

Column A is the internal name of the field.
$this.text refers to the title of that page.

Column B is a comment that may help you understand how the field is used.

Column C is the English phrase.

Column D is where you can write or change the phrase in your language.  Please do not change Columns A, B or C.

Note: Menus may have an '&' character in front of a letter to indicate a menu item with a shortcut.
For example, File would be entered as &File.

Note: Any field that is blank, just ignore it.

Note: Any field marked @Invariant, just ignore it.
