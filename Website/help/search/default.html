Search

Aquila WOL includes a search tool to aid in configuring the software.  Click on Tools -> Search for machines.

Use of the search tool is optional, and it isn't perfect, but it can help you find addresses easily in large networks. It is primarily designed to work with Windows hosts, and you will get an "RPC" error on Linux servers and other hosts that don't answer with WMI (Windows Management Instrumentation).  This is normal.

To use the tool, enter a starting and ending IP address and click "Begin Search". The software will attempt to find hosts, and if possible determine their MAC addresses. Check the check-boxes on hosts you are interested in, and click OK to add them to your database.
