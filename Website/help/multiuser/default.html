Using WOL in a multiuser environment

You can put your WOL database on a shared drive so that multiple users can have access to it.  You can specify the location of the database, which is named "machines.xml", in the WOL options page.

The software is not yet fully written to work in a multiuser environment, with record locking and such, but with reasonable care, you use it this way.
