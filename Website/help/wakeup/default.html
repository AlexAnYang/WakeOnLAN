Waking up a host

To wake-up a host, right click the host and click the "Wake Up" option.

As long as the host icon is highlighted in the window, WOL will ping it to let you know if it is online or offline.  When a host changes state, a notification sound will be heard.  When you wake up a computer, you need to allow sufficient time for it to boot up (a few minutes) before you will see it On-Line and ready in WOL.

You may select multiple hosts at one time to wakeup.
